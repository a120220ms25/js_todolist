# js_todolist


## [Demo](https://a120220ms25.github.io/js_todolist/)


## 簡介
以JavaScript所撰寫的todolist以及時鐘。
可在切換成已完成、待完成項目，下方可顯示尚有多少待完成項目、清除已完成項目。
記錄待完成事項的同時旁邊有時鐘可觀看當前時間，資料存取於localstorage，並不會因為瀏覽器的關閉而遺失紀錄。

## 主要使用技術
* JS、JQuery
